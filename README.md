# Desktop Ansible

Automates the installation and configuration of a new laptop/desktop running POP!\_OS.

## Prerequisites

- Have an SSH key setup with the dotfiles git repo.

```bash
git clone git@gitlab.com:hmajid2301/ansible-desktop.git
cd ansible-desktop
make setup_machine
```
